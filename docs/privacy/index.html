Data Privacy Statement

Effective date:  2025-11-04

1. We do not collect, store, or process any personal data from visitors to this website. The site does not contain: Link to heading

registration or login forms,
contact‑form fields that request personal information,
newsletter sign‑up,
e‑commerce checkout, or
any other mechanism that would capture names, email addresses, phone numbers, IP addresses,
cookies, or similar identifiers.

2. Technical data & server logs Link to heading

Our web host automatically records minimal technical information (e.g., server logs, IP address, timestamp) solely for the purpose of maintaining the availability and security of the site. This data is not linked to any individual, is retained only for a short period (typically 30 days), and is not used for profiling, analytics, or marketing.

3. Legal basis Link to heading

Because no personal data is collected, none of the GDPR legal bases (consent, contract, legal obligation, vital interests, public task, legitimate interests) apply to our processing activities.

4. International transfers Link to heading

No personal data is transferred outside the European Economic Area (EEA) because no such data is collected.

5. Retention period Link to heading

Technical log data described in section 3 is automatically deleted after the retention period set by the hosting provider (normally ≤ 30 days). No personal data is retained.

6. Your rights Link to heading

Since we do not hold any personal data about you, there are no GDPR data‑subject rights to exercise in relation to this website. Should you have any concerns about the brief technical logs mentioned above, you may contact us using the details in section 1.

7. Cookies & tracking technologies Link to heading

The site does not place any cookies, web beacons, or similar tracking technologies on your device. Any essential cookies required for basic server operation (e.g., session handling) are strictly technical and do not identify you personally.

8. Security measures Link to heading

We employ standard industry security practices (TLS/HTTPS encryption, regular server updates) to protect the integrity of the website and any incidental technical data.

9. Changes to this statement Link to heading

If the scope of the website changes and we begin to collect personal data, we will update this statement accordingly and publish the revised version prominently.
